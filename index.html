Step by Step

Step 1

Go to RapidAPI website https://rapidapi.com/hub and create your account

Step 2

Login with the account you created before and click on "My Apps" option, once here we are going to create a new app clicking under the "Add New App" option on the left panel.

Step 3

After we have created our New App, we are able to see our API Key which is completely personal and our access to this API

Step 4

Then, we are free to go and look for any API inside RapidAPI, there are a lot of free APIs but there are also private/subscription APIs.
For this assignment I am going to use "Free-to-Play Games Database" API, which shows the Free-to-Play games for different kind of consoles, information, categories, and so on.

Step 5

After adding the code from the right panel to our JavaScript, we can adapt it however we want.
